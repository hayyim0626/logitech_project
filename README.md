## 🖥 &nbsp; 프로젝트 소개
- 주제: 로지텍 공식 홈페이지 클론 프로젝트
- 프로젝트 기간: 2020.09.14 - 2020.09.25 (12일 간)
- 협업 : Git, Trello, Slack 활용


## 🛠 &nbsp; 기술 스택
- Front-end
  - HTML, CSS
  - JavaScript(ES6+)
  - React 
  - Slick.js (library)
  - SCSS
  - AWS(EC2)


- Back-end
  - 언어: python
  - 프레임워크: django
  - DB: MySQL
  - 암호화: jwt, bcrypt
  - 크롤링: beautifulsoup, selenium
  - 배포: AWS
